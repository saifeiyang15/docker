## 🎨 设计特点

### 1. 唯美配色方案

#### 主色调
- **渐变背景**: `linear-gradient(135deg, #a8edea 0%, #fed6e3 100%)`
  - 薄荷绿到粉色的柔和过渡
  - 营造梦幻、温馨的氛围

#### 辅助色彩
- **卡片背景**: 白色到淡粉色渐变
- **按钮**: 薄荷绿到粉色渐变
- **高亮**: 金色光晕效果

### 2. 视觉效果

#### 动画效果
1. **背景动画**
   - 浮动的网格点阵
   - 20秒循环动画
   - 增加空间感和层次感

2. **骰子动画**
   - 旋转效果（roll animation）
   - 0.1秒快速旋转
   - 增强掷骰子的真实感

3. **角色动画**
   - 弹跳效果（bounce）
   - 0.5秒弹跳动画
   - 移动时的流畅过渡

4. **光晕效果**
   - 角色周围的呼吸光晕
   - 3秒循环动画
   - 营造魔法氛围

5. **脉冲效果**
   - 当前位置的脉冲高亮
   - 2秒循环动画
   - 清晰标识玩家位置

6. **闪烁效果**
   - 任务奖励的星星闪烁
   - 2秒交替动画
   - 增加趣味性

#### 视觉细节
1. **毛玻璃效果** (`backdrop-filter: blur()`)
   - 卡片和弹窗使用毛玻璃背景
   - 增加现代感和层次感

2. **多层阴影**
   - 使用多个 box-shadow 叠加
   - 创造立体感和深度

3. **渐变边框**
   - 使用伪元素创建渐变边框
   - 增加精致感

4. **圆角设计**
   - 统一使用较大的圆角（15-30px）
   - 柔和、友好的视觉体验

## 🎨 设计亮点

### 1. 色彩心理学应用

- **薄荷绿**: 清新、舒适、放松
- **粉色**: 温柔、梦幻、可爱
- **白色**: 纯净、简洁、优雅
- **金色**: 珍贵、奖励、成就

### 2. 动画设计原则

- **缓动函数**: 使用 `cubic-bezier(0.4, 0, 0.2, 1)` 实现自然过渡
- **时长控制**: 0.3-0.4秒的过渡时间，既不拖沓也不突兀
- **循环动画**: 2-3秒的循环周期，不会让人感到疲劳

### 3. 视觉层次

1. **背景层**: 渐变 + 动画网格
2. **内容层**: 毛玻璃卡片
3. **交互层**: 按钮、表单
4. **反馈层**: 悬停效果、动画

### 4. 用户体验优化

- **即时反馈**: 所有交互都有视觉反馈
- **清晰导航**: 面包屑、返回按钮
- **错误处理**: 友好的错误提示
- **加载状态**: 加载动画和提示

## 🎯 API 接口

### 任务管理接口

```
GET    /api/game/tasks          - 获取所有任务
GET    /api/game/tasks/{id}     - 获取单个任务
POST   /api/game/tasks          - 创建任务
PUT    /api/game/tasks/{id}     - 更新任务
DELETE /api/game/tasks/{id}     - 删除任务
```

### 单人游戏接口

```
POST   /api/game/single/start   - 开始游戏
POST   /api/game/single/roll    - 掷骰子
GET    /api/game/single/state/{userId} - 获取状态
POST   /api/game/single/reset/{userId} - 重置游戏
```
